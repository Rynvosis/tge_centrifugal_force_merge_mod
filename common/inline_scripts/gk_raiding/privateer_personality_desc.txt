# Privateer personality triggered desc blocks
# Generates one desc per personality, gated by country flag on $SCOPE$
# Expects loc keys: $KEY$.stoic, $KEY$.screamer, $KEY$.cultist, $KEY$.calculating, $KEY$.merchant, $KEY$.automaton
#
# Usage:
# inline_script = {
#     script = gk_raiding/privateer_personality_desc
#     KEY = gk_raiding_privateers.102.desc
#     SCOPE = event_target:gk_privateer_enclave_country
# }
